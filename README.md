# Vagas de Matemática – Minas Gerais

Transformação de dados públicos em informação estruturada para análise de oportunidades reais.

## Contexto

Quem foi aprovado no concurso para Professor de Matemática em Minas Gerais rapidamente se depara com uma dificuldade:

as informações sobre vagas estão espalhadas em diversos documentos, principalmente PDFs, sem padronização e com baixa clareza.

Embora o edital apresente vagas por cidade, os detalhes completos estão distribuídos em múltiplos arquivos do processo de designação online, dificultando a análise e a tomada de decisão.

## Problema

Os dados disponíveis apresentam diversos desafios:

- informações fragmentadas em múltiplos PDFs  
- ausência de padronização  
- dificuldade para consolidar os dados  
- baixa visibilidade sobre o total real de vagas  

Na prática, isso torna difícil responder perguntas simples, como:

quantas vagas realmente existem?  
em quais escolas?  
em quais turnos?  
qual o tipo de vaga disponível?

## Solução

Foi desenvolvido um processo para transformar dados não estruturados em informação clara e analisável.

O projeto envolveu:

- extração de dados a partir de PDFs públicos  
- tratamento e padronização das informações  
- consolidação em uma base estruturada  
- criação de um painel para visualização  

O resultado é uma visão organizada das vagas de Matemática, permitindo análise rápida e objetiva.

## Resultado

A solução permite visualizar:

- total de vagas disponíveis  
- distribuição entre cargo vago e substituição  
- vagas por escola  
- vagas por turno  

Exemplo de resultado:

Total de vagas: 172  
Cargos vagos: 136  
Substituições: 36  

## Tecnologias utilizadas

Python  
Pandas  
ETL (Extração, Transformação e Carga de dados)  
Excel  
Power BI (ou ferramenta de visualização utilizada)

## Impacto

Este projeto transforma um processo manual, disperso e pouco acessível em uma análise estruturada e clara.

Mais do que organizar dados, a solução permite:

- melhor compreensão do cenário de vagas  
- apoio à tomada de decisão  
- acesso simplificado à informação pública
